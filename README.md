# AMAS - Mobility AI Assistant

## Requirements
- Python 3.8+
- uv (https://github.com/urpylka/uv)

## Installation
1. Create and activate virtual environment:
   ```bash
   uv venv --py=python3.8
   uv source
   ```
2. Install dependencies:
   ```bash
   uv install
   ```

## Usage
Run the application:
```bash
uv run main.py
```
